#使用
gitbook提供[可视化编辑器(GitBook Editor)](https://www.gitbook.com/editor)及[命令行工具gitbook-cli](https://github.com/GitbookIO/gitbook-cli)。

###安装需求
* NodeJS (v4.0.0 +)
* Windows, Linux, Unix, or Mac OS X

###npm安装
`npm install -g gitbook-cli`

###帮助
`gitbook help`

###初始化
`gitbook init [book]`

###构建
`gitbook build [book]`

###构建pdf
`gitbook pdf [book]`

###预览
`gitbook serve [book]`

###安装依赖
`gitbook install [book]`
